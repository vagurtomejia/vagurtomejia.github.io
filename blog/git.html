What are the benefits of version control? When your projects are managed using a version control system, here are some advantages that you have:
  • everyone knows what the other members of the team are doing
  • you can have a control of permissions
  • you can store the different succesives versions of the project
  • you can store detailed information on the logs of different versions
  • keep track of all the changes, so that you could go back to a working version at any time
  • you can experiment without messing up the final product
How does git help you keep track of changes?
  • every commited changes have an associated message wich provides information on the changes
  • every commit is a check point
  • every changes are associated with an user
  • we have a list of all the files that where changed at a specific point
  • it´s possible to compare two different versions to see the changes
Why use GitHub to store your code?
  • it uses Git VCS (Version Control System)
  • it provides with remote repositories available at any time
  • it has features and tools usefull for collaborative work
  • it´s also a social coding website
