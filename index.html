HTML review of the CNVC site

Here is my technical review of the CNVC site:

  • The type of the website is XHTML 1.0 Strict.
  • It gives 55 errors and 8 warnings in the W3C Markup Validation Service.
  • There are almost any elements that gives semantic meaning to the content of the site.
  • lement itself is not supported in the document type you are using, as an undefined element will have no supported attributes; in this case, see the element-undefined error message for further information.
